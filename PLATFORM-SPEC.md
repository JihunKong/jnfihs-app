# 전남미래국제고 종합 플랫폼 (JNFIHS Platform)

## 📋 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | JNFIHS Platform (전남미래국제고 통합 플랫폼) |
| **플랫폼** | 크롬 기반 웹앱 (PWA) - 크롬북 최적화 |
| **대상** | 학생 50명, 교사, 학부모 |
| **지원 언어** | 한국어, 몽골어, 러시아어, 베트남어 |
| **개교일** | 2026년 3월 |

### 1️⃣ 학생-교사 메신저

#### 기능 상세

| 기능 | 설명 |
|------|------|
| 1:1 채팅 | 학생 ↔ 담임/교과 교사 |
| 그룹 채팅 | 학급, 동아리, 프로젝트별 |
| 실시간 번역 | 메시지 전송 시 자동 번역 |
| 읽음 확인 | 메시지 읽음 표시 |
| 파일 공유 | 이미지, 문서, 과제 첨부 |
| 알림 설정 | 수업 시간 방해 금지 모드 |

#### 화면 구성

```
┌─────────────────────────────────┐
│ ← 김영희 선생님        [번역:ON] │
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────┐        │
│  │ 오늘 과제 제출했나요?  │  10:30 │
│  │ ─────────────────── │        │
│  │ 今日の課題を提出しましたか？│        │
│  └─────────────────────┘        │
│                                 │
│        ┌─────────────────────┐  │
│  10:32 │ 네, 방금 제출했습니다   │  │
│        │ ─────────────────── │  │
│        │ Тийм, би дөнгөж илгээсэн │  │
│        └─────────────────────┘  │
│                                 │
├─────────────────────────────────┤
│ [📎] 메시지 입력...        [전송] │
└─────────────────────────────────┘
```

#### 데이터 구조

```sql
-- 채팅방
CREATE TABLE chat_rooms (
  id UUID PRIMARY KEY,
  type VARCHAR(20), -- 'direct', 'group', 'class'
  name VARCHAR(100),
  created_at TIMESTAMP
);

-- 메시지
CREATE TABLE messages (
  id UUID PRIMARY KEY,
  room_id UUID REFERENCES chat_rooms(id),
  sender_id UUID REFERENCES users(id),
  content TEXT,
  content_ko TEXT,      -- 한국어 번역
  content_original TEXT, -- 원문
  locale VARCHAR(5),     -- 원문 언어
  created_at TIMESTAMP
);
```

#### 기능 상세

| 기능 | 설명 |
|------|------|
| 가정통신문 | 학교 공지 발송 (자동 번역) |
| 활동 갤러리 | 학교 행사 사진/영상 공유 |
| 상담 예약 | 학부모 상담 일정 예약 |

| 긴급 연락 | 긴급 상황 즉시 알림 |

#### 가정통신문 자동 번역 Flow

```
교사가 한국어로 작성
        ↓
    [발송 버튼]
        ↓
Claude API로 4개 언어 번역
  (한국어, 몽골어, 러시아어, 베트남어)
        ↓
학부모별 설정 언어로 전송
        ↓
앱 푸시 + 이메일 발송
```
